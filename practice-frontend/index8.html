Semantic HTML Tags

Semantic tags give meaning to the structure of web content

Navigation <nav>

Used for navigation menus and links

  • Header
  • Main Content
  • Footer

Main Content <main>

Contains the main content of the document

Article <article>

Represents a self-contained composition (like a blog post, article, or comment)

Section <section>

Groups related content together

Aside <aside>

Contains content that is tangentially related to the content around it (like sidebars)

Other Common Semantic Tags

  • <figure> and <figcaption> - For images with captions
  • <time> - For dates and times
  • <mark> - For highlighted text
  • <details> and <summary> - For expandable details
